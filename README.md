# Teledyne_ScopeFoundry
Code for the controlling and acquiring from the Teledyne camera Retiga E7, developed with PVCAM, Programmable Virtual Camera Access Method library (other supported hardwares are  all modern Teledyne Photometrics brand hardware together with Teledyne QImaging cameras such as Retiga R1, R3, R6, LUMO, ELECTRO, MicroPublisher 6.0 and optiMOS). The wrapper can be installed at this [link](https://github.com/Photometrics/PyVCAM/tree/master).
